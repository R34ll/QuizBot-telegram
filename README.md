# Overview
The Quiz Telegram Bot is a simple yet powerful bot that allows users to play quizzes fetched from the OpenTDB (Open Trivia Database) API right within their Telegram chat. It provides an interactive and engaging way for users to test their knowledge on various topics.

# Features
- Quiz Categories: Users can choose from a variety of quiz categories.
- Difficulty Levels: Quizzes come with different difficulty levels - easy, medium, and hard.
- Score Tracking: The bot keeps track of users' scores and displays them at the end of each quiz.


# Getting Started
## Prerequisites
- Python 3.x
- Telegram Bot Token
- pyTelegramBotAPI
- OpenTDB API Key
